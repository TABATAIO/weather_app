# 天気予報アプリ フロントエンド

## 概要

天気予報と連動するマスコット育成アプリのフロントエンド部分です。
マスコットとの会話、天気に応じた背景変更、ミッション機能などを提供します。

## 機能

### 📱 マスコットページ

- **インタラクティブマスコット**: タップでアニメーション
- **天気連動背景**: 天気データに応じた背景の動的変更
- **AI会話機能**: マスコットとの自然な会話
- **ステータス管理**: レベル、満腹度、気分の表示
- **ミッションシステム**: 日次タスクと報酬システム

### 🎨 レスポンシブデザイン

- モバイルファーストデザイン
- iPhone/Android対応
- タッチインタラクション最適化

## 使用方法

### 開発環境での起動

```bash
# バックエンドサーバー起動
cd Backend
npm start

# フロントエンドはLive Serverや任意のHTTPサーバーで起動
# 例：VS CodeのLive Server拡張機能を使用
```

### APIエンドポイント

- **チャット**: `POST /api/mascot/chat`
- **天気データ**: `GET /api/weather/city/:city`
- **マスコットステータス**: `GET /api/mascot/status`
